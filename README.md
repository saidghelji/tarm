
## 🚀 Project Overview

TARM is a responsive business website created to showcase renovation services and build trust with local clients:
- Multi-language support
- Service portfolio
- Project gallery
- Client testimonials
- Contact form lead generation

## 🧩 Features

- French and Arabic language support
- Service portfolio and project showcase
- Lead generation contact form
- Testimonials section
- Mobile-responsive layout
- Local SEO optimization

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- EmailJS
- Lucide React
- Internationalization
